Gere o prompt do Higgsfield para um ESTUDIO DE TATUAGEM.

Carregue a skill `gerador-prompt-cinematografico` e use o mapa de jornada "Estudio de tatuagem"
da referencia `jornadas-por-nicho.md`.

Pergunte apenas o essencial que faltar: nome do estudio, nome do tatuador, estilos, cor de
destaque, e se e para TRANSFORMAR um site existente na pasta ou CRIAR DO ZERO a partir de uma URL.

Entregue SOMENTE o prompt final, num unico bloco de codigo copiavel, salvo em outputs. Sem
roteiro, sem explicacao longa.
