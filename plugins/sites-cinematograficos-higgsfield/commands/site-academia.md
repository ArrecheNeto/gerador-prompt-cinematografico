Gere o prompt do Higgsfield para uma ACADEMIA ou ESTUDIO FITNESS.

Carregue a skill `gerador-prompt-cinematografico` e use o mapa de jornada "Academia / estudio
fitness" da referencia `jornadas-por-nicho.md`.

Pergunte apenas o essencial que faltar: nome da academia, nome do treinador/aluno que aparece no
reveal, modalidade, cor de destaque, e se e para TRANSFORMAR um site existente ou CRIAR DO ZERO
a partir de uma URL.

Entregue SOMENTE o prompt final, num unico bloco de codigo copiavel, salvo em outputs. Sem
roteiro, sem explicacao longa.
